The project addresses a common issue where violations are incorrectly registered under the vehicle owner instead of the actual driver, leading to legal disputes and unfair penalties.

The system uses advanced deep learning models and computer vision techniques to analyze driver images captured by traffic monitoring cameras. By applying Convolutional Neural Networks (CNNs) along with face recognition models such as **FaceNet** and **ArcFace**, Eye-In compares detected faces with a secure database to determine identity with high accuracy. Similarity metrics including cosine similarity and Euclidean distance are used to validate matches.

Eye-In integrates with existing traffic systems through secure APIs, enabling automated violation assignment while reducing manual processing. The system also includes a transparent objection mechanism that allows violations to be reanalyzed automatically, improving fairness and accountability.

Designed to support smart city initiatives and Saudi Vision 2030, the solution enhances traffic management efficiency, reduces administrative workload, increases public trust, and promotes accurate law enforcement through AI-driven automation.
